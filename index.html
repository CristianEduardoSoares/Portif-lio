Minha Biblioteca: Uma Webpage Personalizada

O projeto consiste em uma página web desenvolvida com HTML e CSS, projetada para funcionar como uma livraria virtual, com foco na exibição de livros favoritos e em funcionalidades típicas de e-commerce. O principal objetivo é criar um ambiente onde o usuário possa explorar e interagir com sua coleção pessoal de livros, adicionando itens à estante, marcando como favoritos, comprando e recebendo recomendações personalizadas.

O layout é responsivo, utilizando uma meta tag viewport e flexbox para garantir uma boa experiência em diferentes dispositivos. A escolha da tipografia e do esquema de cores visa criar uma atmosfera acolhedora, incentivando os usuários a explorar a coleção de livros e facilitar a compra online.

Além disso, o projeto inclui a personalização de uma área de perfil, onde o usuário pode gerenciar suas preferências e acompanhar suas compras. As seções de recomendação, como "Favoritos", "Mais Vendidos" e "Novos Lançamentos", ajudam a direcionar o usuário para títulos relevantes, enquanto a integração com o carrinho de compras e o sistema de pagamento torna o processo de compra simples e direto.

Minha Biblioteca: Uma Webpage Personalizada

Este projeto de flashcards foi desenvolvido para auxiliar alunos, especialmente do ensino médio, que buscam uma revisão de conceitos básicos de química orgânica. A aplicação utiliza a metodologia de flashcards para revisar conceitos essenciais, como hidrocarbonetos, anéis aromáticos, a classificação de carbonos e cadeias carbônicas, e a fórmula de bastão. Esses flashcards são uma ferramenta essencial para o estudo e a preparação para provas de química.

O projeto utiliza as seguintes tecnologias: HTML para a estruturação do conteúdo da página, incluindo as seções de perguntas e respostas; css para a estilização da aplicação, garantindo um layout responsivo e atraente; e JavaScript para adicionar interatividade, permitindo a navegação entre os flashcards e a revelação das respostas.

Os flashcards cobrem tópicos essenciais da química orgânica, como: a definição e classificação de hidrocarbonetos, a estrutura dos anéis aromáticos com ligações duplas alternadas, a classificação dos carbonos como primário, secundário, terciário ou quaternário, a diferença entre cadeias carbônicas normais e ramificadas, e a interpretação da fórmula de bastão, que representa os átomos de carbono.

Este projeto foi desenvolvido pela Alura, sem fins lucrativos, com o objetivo de fornecer um recurso educacional acessível para estudantes interessados em aprofundar seus conhecimentos na área de química orgânica.

Minha Biblioteca: Uma Webpage Personalizada

Este projeto de apresentação da Barbearia Alura foi desenvolvido para destacar os serviços e valores da barbearia, localizada no centro da cidade. Utilizando HTML, CSS e JavaScript, o site estrutura o conteúdo, cria um layout responsivo e oferece interatividade entre as seções, como a missão da empresa, seus benefícios e os serviços oferecidos.

O objetivo do projeto é apresentar a Barbearia Alura, destacando sua fundação em 2019 e sua missão de promover autoestima e qualidade de vida aos clientes. São também enfatizados os principais benefícios, como atendimento especializado, espaço diferenciado, localização estratégica e profissionais qualificados.

Este projeto foi desenvolvido com fins educacionais e não é um site real. Ele tem como propósito proporcionar uma experiência de aprendizado sobre o desenvolvimento de páginas web, utilizando HTML, CSS e JavaScript de forma prática. O site é um exemplo fictício, criado para fins de estudo, e não representa uma barbearia real.

O projeto serve como uma ferramenta de ensino, ideal para quem deseja aprender as técnicas de criação de sites, praticando habilidades de desenvolvimento web em um contexto prático e sem fins comerciais.
